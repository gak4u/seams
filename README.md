# Sea-M-S

Sea-M-S is a CMS (Content Management System) based on SvelteKit, inspired by the popular Payload CMS. It provides a simple and intuitive interface for managing your content and publishing it to your website.

## Getting Started

To get started with Sea-M-S, you can follow these steps:

1. Clone the Sea-M-S repository to your local machine:
```bash
git clone https://github.com/your-username/sea-m-s.git
```

2. Install the dependencies using npm:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to [http://localhost:5173](http://localhost:5173) to see the Sea-M-S interface.

## Features

Sea-M-S provides the following features:
- Simple and intuitive interface for managing your content
- Support for multiple content types and fields
- Customizable templates for displaying your content on your website
- User authentication and authorization
- Integration with MongoDB

## Contributing

If you want to contribute to Sea-M-S, you can follow these steps:
1. Fork the Sea-M-S repository to your GitHub account.
2. Clone your forked repository to your local machine.
3. Create a new branch for your changes.
4. Make your changes and commit them to your branch.
5. Push your branch to your forked repository on GitHub.
6. Create a pull request from your branch to the main Sea-M-S repository.
